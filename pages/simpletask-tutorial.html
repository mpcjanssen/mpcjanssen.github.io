Basic rules & procedures

Enter a new task:

Create the task

Open the app, look at the top panel for the well known '+' symbol. Tap on that '+' symbol

Task Syntax

Syntax to indicate: List, Tag, Threshold (Start) date, Due (Finish) date, Recurrance, Priority.

List= @EXPRESSION,
Tag= +EXPRESSION,
Threshold date= t:YYYY-MM-DD,
Due date= due:YYYY-MM-DD,
Recurrence= rec:0-9d/w/m/y,
Priority= (ENGLISH CAPITAL LETTER).

An example task:

Clean study room at home weekly, having less priority.

The task in the editting box should look like (while the task is entered on 2013-12-13):

2013-12-13 (D) Clean +studyroom at @home due:2013-12-17 t:2013-12-15 rec:1w

It's not necessasary to use all tags in a single task as mentioned in this example.

Although it does not clutter while viewing the task in the task list, you can turn off auto insertion of task creation date from the App Settings dialog. Beware, this will loss that filter.

Usually Tag is subordinate to List, although they may be independent. It's better to use 'List' for greater contexts e.g., project, place, person and 'Tag' for lesser contexts e.g., subproject, activity. There may be more than one context in a task.

You can write due time also, although there is no dedicated filter for it, so no special syntax.

To avoid confusion & to speed up your editing, some buttons are available below, to enter different tags easily. Just tap the required. In the LISTS & TAGS dialog either select existing one or enter a new one as you need.

After writing the task, remember to save it tapping the well known big "right mark" at top-right corner. Or, cancel/delete the task tapping the well known big "X" at top-right corner.

View the task list:

There are 4 ways:

Using the Android Launcher

Home -> App Menu -> scroll to the app icon & tap

Using a shortcut

If privacy is less required. Place the app shortcut on Home screen. Just tap it. App shortcut can be done 2 ways:

  • One is the conventional way for Android.
  • Another way is, if using a saved filter regularly, long press it in the drawer, in the popup menu, select "To homescreen".

Using the app widget

If privacy is not required. Place the app widget on Home screen. You can use that widget or open the full app from that. While creating the widget it will ask for preferred filters. Do as required.

The widget is configurable. Look for the related buttons at the interface & app setting dialog.

Using the calendar

If privacy is less required. Get notification alert through the installed Calendar app. Open the notice by taping. It will only show the notified task in details.

To edit a task:

Tap the task to activate task editting buttons on top panel. Then do any of the following:

To delete the task:

Tap Action bar -> select the Delete option

Edit the whole task:

Tap the pen symbol. After editting remember to save the editting by tapping the "right mark" symbol at top-right corner. Or delete the task tapping the "X" at top-right corner.

To get out without editting tap the back button.

Just update Lists or Tags:

Tap the list or tag symbol.

Just update Priority:

Tap the star.

Just update threshold/due date:

Tap Action bar -> select the required option

Get a reminder notice:

This app natively doesn't support notification. It can be done using a Calendar app.

First (install & then) make sure that Google Calendar is working well.

Next open this app, tap the required task to task editting buttons on top panel. Tap Action bar -> select Add reminder

Fill the required data in the poped up dialog from the Calendar app.
