Building fossil for Android

di 23 april 2013

To build fossil for Android, the easiest way is to use the Android NDK. The changes needed to build fossil on a Linux machine can be found at http://mpcjanssen.nl/fossil/fossil/timeline?r=build-android

After running the build.sh script, the Android fossil binary can be found in the obj subdirectory. This fossil binary works on Android and even allows you to host fossil repos from your Android device.

A prebuilt binary can be downloaded from http://mpcjanssen.nl/files/android/fossil
